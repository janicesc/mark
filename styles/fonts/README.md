# Fonts

Place your **Roobert** web font files here so the site uses Roobert as the theme font:

- `Roobert-Regular.woff2` (weight 400)
- `Roobert-SemiBold.woff2` (weight 600)
- `Roobert-Bold.woff2` (weight 700)

If these files are missing, the site falls back to **DM Sans** (loaded from Google Fonts).

You can export WOFF2 from your Roobert license (e.g. from Displaay or your type provider) or use a subset for the web.
